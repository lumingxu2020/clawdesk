# ClawDesk

macOS 上的 OpenClaw 图形化管理工具。

## 功能

- 📊 **系统概览** - CPU/内存实时监控
- ⚡ **进程管理** - Gateway/Node 状态与重启
- ⚙️ **配置编辑** - JSON 可视化编辑器
- 🧠 **记忆管理** - 查看/清理记忆文件
- 🧩 **技能市场** - 安装/管理 Skills
- ⚡ **快捷命令** - 一键执行常用操作

## 安装

```bash
# 从 DMG 安装
双击 ClawDesk-1.0.0-arm64.dmg

# 或直接运行 App
双击 ClawDesk.app
```

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm start

# 构建
npm run build
```

## 技术栈

- Electron 33
- React
- electron-builder

## 系统要求

- macOS 12+ (Apple Silicon)
